Cell Fracture

Process to cell fracture object - video

  1. assign 2 materials (outside and inside)
  2. enable Cell Fracture add-on
  3. Use settings similar to this
    1. The Material setting to 1 = enables outside and inside materials
    2. Set Noise to 1
    3. Source Limit 200 (amount of fractures that will happen)
  4. Initial Result
  5. Move fractured pieces into new collection called “Cells”
  6. Create Plane and move it to Default Collection
  7. With Plane selected add Geometry Nodes to it.
  8. Create Collection Info node and choose Cells collection
    1. set to Relative and Separate Children
  9. Hide Cells collection so now only Geometry Nodes instance is visible
  10. Add Scale Instances, Rotate Instances and Translate Instances nodes for versatility
  11. Add Empty to the scene and move it to the side (this will control Scale, Rotate, Translate instances)
  12. Drag Empty from outliner into Geometry Nodes graph
    1. Change Empty Object Info node to Relative
  13. Create Position node (to track position of Empty), Math Node set to Distance and Map Range node (to keep things constrained)
    2. Adjust Map Range settings to get a disintegrate effect as Empty moves through
